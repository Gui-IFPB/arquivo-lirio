# 🌹 Arquivo Lirio

> **"Decifra-me, ou te devoro."**

Uma experiência interativa de investigação inspirada em enigmas clássicos da internet.

Este projeto apresenta uma pequena narrativa investigativa onde o visitante assume o papel de um **observador** tentando entender eventos estranhos escondidos em arquivos aparentemente comuns.

Nem tudo está visível.

Algumas coisas precisam ser **descobertas**.

---

## 🕵️ A Investigação

Durante a navegação você encontrará:

- pistas escondidas
- páginas conectadas
- um terminal investigativo
- um arquivo criptografado
- elementos simbólicos

A história gira em torno de um caso conhecido apenas como:

### **Registro 27**

Alguns dizem que foi um acidente.  
Outros dizem que nunca deveria ter sido encontrado.

---

## 🔎 Como explorar

1. Comece pela página inicial.
2. Observe os detalhes.
3. Leia o código fonte quando necessário.
4. Teste comandos no terminal.
5. Siga as pistas.

Se você chegou até aqui, talvez já tenha percebido:

> **O lírio não floresce sem motivo.**
